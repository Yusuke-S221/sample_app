アプリ概要
--------------------------------------------------------------------------
Travel Hubは、旅行者をメインターゲットにしたブログアプリです。

機能、できること
--------------------------------------------------------------------------
・記事投稿、編集、削除機能  
・記事一覧、詳細表示機能  
・画像投稿、編集機能  
・ユーザー登録機能  
・ログイン機能  
・パスワード暗号化   
・アクセス制限機能  
・ページネーション機能
・いいね機能   
・単体、統合テスト

技術、どうやって作ったか
--------------------------------------------------------------------------
・Rails  
・Heroku(デプロイ)    
・sqlite3(開発、テスト) PostgreSQL(本番)    
・bcrypt(パスワード暗号化)  
・kaminari(ページネーション)
